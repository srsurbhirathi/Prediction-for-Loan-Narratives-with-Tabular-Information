# Prediction-for-Loan-Narratives-with-Tabular-Information
This focuses on using Machine learning algorithms like XGBoost, Logistic Regression, BERT to predict whether loan narratives written by borrowers add more predictive power in comparison to just using tabular structured infromation. The dataset I have used is Lending club dataset from Kaggle  

Technologies:
Programming Language: Python
Libraries: Pandas, Scikit-learn, Matplotlib, Seaborn, BERT 

Visualization: plotly

Data Source:
https://www.lendingclub.com/info/download-data.action
